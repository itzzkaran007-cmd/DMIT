Welcome to DMIT

Dermatoglyphics Multiple Intelligence Test (DMIT) is a study of fingerprint patterns to analyze cognitive strengths and learning styles. This website is a hackathon prototype for demonstrating how fingerprint-based insights could be visualized.

Features

  • Upload and preview fingerprints
  • Mock analysis engine
  • Interactive radar chart visualization
  • Export report as PDF
